# Product Overview

- Web上の商品最安値検索・相場把握ツール
- 複数のECサイト・フリマサービスを横断的にクロールし、価格情報を集約

## Core Capabilities

- 商品価格のクロール・収集
  - ヤフオク、ヤフーフリマ、メルカリ、じゃんぱら、イオシス、パソコン工房、リコレ（中古ソフマップ）
- 相場分析・日次サマリー
  - プラットフォーム別の日次購入統計
- カテゴリ・商品管理
  - 階層構造カテゴリ（closure_tree）
  - クロール設定（キーワード、価格帯、除外条件）

## Target Use Cases

- 中古商品の相場調査
- 最安値アラート・監視
- 価格推移トレンド把握

## Value Proposition

- 複数プラットフォームを一元管理
- 自動クロールによるリアルタイム価格更新
- 学習プロジェクトとしてのRails/TypeScriptキャッチアップ
- 運用監視（Datadog、BugSnag）の実践
